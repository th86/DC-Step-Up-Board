DC-DC Step-Up Board
=========

This is an NE555-based adjustable DC-DC step-up board, which converts 9~12Vdc to up to 400Vdc. 

WARNING: THE HIGH VOLTAGE IS LETHAL, USE IT WITH CAUTION!
